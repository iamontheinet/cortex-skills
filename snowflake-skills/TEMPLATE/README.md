## Usage

### With Cortex Code

```json
{
  "remote": [
    {
      "source": "https://github.com/Snowflake-Labs/snowflake-ai-kit",
      "ref": "main",
      "skills": [{ "name": "my-skill-name" }]
    }
  ]
}
```

### With Other Agents

Copy the `SKILL.md` file into your agent's rules directory. See the [main README](../../README.md) for per-agent instructions.
